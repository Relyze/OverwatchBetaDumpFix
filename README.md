# Usage:
1. Compile the solution and copy the "OverwatchDumpFix.dp64" file to "x64dbg/plugins/OverwatchDumpFix.dp64"
2. Launch x64dbg and GameClientApp.exe, attach x64dbg to the process, and execute the OverwatchDumpFix command.
3. After verifying the command has completed successfully in the x64dbg log window, you may utilize Scylla to dump the binary.
